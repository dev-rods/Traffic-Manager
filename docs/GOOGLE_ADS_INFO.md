Como funciona a autorização para gerenciamento no google ADS?

Nossos clientes -> Tem uma conta de anunciante comum

A gente -> Tem uma conta MCC. 
    Pegamos o ID do cliente e solicitamos permissão para gerenciar a conta do cliente.
    O cliente tem que aceitar a solicitação.
    Depois de aceito, a gente pode gerenciar a conta do cliente.

A conta MCC possui um Token de desenvolvedor.
Atualmente, temos o nível de acesso "Conta de teste"
